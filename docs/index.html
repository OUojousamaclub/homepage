大阪大学お嬢様部とは

2020年頃、Twitter上でお嬢様言葉を使う「お嬢様部」というミームが全国の大学や高校に次々と生まれるという不思議な現象が起こりました。 約1年後、わたくしたちはTwitterを飛び出して画面のこちら側で活動を行うことを始め、サークルとしての「大阪大学お嬢様部」が誕生しました。
わたくしたち大阪大学お嬢様部は「『お嬢様』とは心の在り方。出自も性別も年齢も一切関係なく、高貴な心を持っていれば誰でも『お嬢様』ですわ」を理念に、高貴な心を磨くため日々活動しています。
